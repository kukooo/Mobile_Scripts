# Mobile Scripts
Some script that are usefull for mobile pentesting
## iOS
* `GetIOsAppsPaths.py` will print all the installed apps, with their version, App Container and Data Container
    * Requirements: `libimobiledevice`
## Android
* `apkpuller` will list all the apps in your Android device and will let you choose to pull all the slices from it and merge them in a unique .apk file along the base.apk
    * Requirements: `java`
